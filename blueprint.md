# Lottery Number Picker

## Overview

A simple web application to generate random lottery numbers.

## Features

*   Generate 6 random numbers between 1 and 45.
*   Display the generated numbers.
*   Button to generate new numbers.

## Plan

1.  **HTML:** Create the basic structure with a title, a container for the numbers, and a button.
2.  **CSS:** Style the elements to be visually appealing.
3.  **JavaScript:** Implement the logic to generate and display the random numbers.
